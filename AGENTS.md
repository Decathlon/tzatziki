# Tzatziki Project Overview

Tzatziki is a comprehensive library of Cucumber step definitions and testing utilities designed to accelerate and simplify Behavior-Driven Development (BDD) for Java and Spring-based applications.

## Purpose

The library provides ready-to-use steps for common infrastructure and application layers, including:
- **HTTP/REST**: Mocking servers and asserting on request/response interactions.
- **Spring Data/JPA**: Managing database state and verifying persistence.
- **Kafka**: Producing and consuming messages for asynchronous testing.
- **OpenSearch/MongoDB**: Integrated steps for NoSQL and search engine validations.
- **Logging**: Asserting on application logs.
- **MCP (Model Context Protocol)**: Support for AI-integrated testing environments.

By using Tzatziki, developers can write expressive, high-level Gherkin scenarios without having to manually implement repetitive infrastructure "glue" code.
