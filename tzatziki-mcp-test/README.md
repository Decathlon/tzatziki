# Tzatziki MCP Test

This module provides Cucumber steps to test MCP (Model Context Protocol) servers.

## Overview

The Model Context Protocol (MCP) is an open standard that enables seamless integration between AI systems and data sources. This module allows you to write Cucumber tests for MCP servers.

## Dependencies

Add this dependency to your project:

```xml
<dependency>
    <groupId>com.decathlon.tzatziki</groupId>
    <artifactId>tzatziki-mcp-test</artifactId>
    <version>1.0.0-SNAPSHOT</version>
    <scope>test</scope>
</dependency>
```
